# ComfyUI-Krea2WashControl

Krea2 洗图本地辅助节点。这个插件是单独的本地插件，不修改 `ComfyUI-ConditioningKrea2Rebalance`，所以正常更新上游 Git 插件不会覆盖这里的节点。

## 节点

- `Krea2洗图重平衡（肤质安全）`：给 Krea2 提供柔和的参考图条件，尽量保构图、姿势和身体比例，同时降低参考 latent 对皮肤噪点的继承。
- `Krea2洗图参考尺寸`：让输出尺寸跟随参考图比例，默认按 8 的倍数对齐。`最大百万像素=0` 时尽量保持原图尺寸。
- `Krea2洗图Latent肤质清理`：在 `VAEEncode` 前轻度清理参考图肤色区域，减少低降噪继承的黑点、暗斑和粗糙纹理。
- `Krea2洗图皮肤暗点清理`：检测肤色区域里的小暗点，并用局部均值轻修，主要用于压制点状扩散和黑色横线感。

## 中文参数

节点参数名已经直接中文化，常调参数包括：

- `参考强度`：越高越保原图姿势和比例，也越容易继承原图脏点。
- `文字结束进度` / `图像开始进度`：控制文字和参考图条件在采样过程中的介入时机。
- `肤质平滑`：肤色区域清理强度，越高越干净，也越容易变糊。
- `暗点阈值`：越低越容易识别小黑点，过低可能误伤阴影或衣物褶皱。
- `全局肤色平滑`：只在肤色区域做轻微整体平滑，用来压残留脏感。

## 推荐链路

当前推荐链路：

`参考图 -> 参考尺寸 -> Latent肤质清理 -> 皮肤暗点清理 -> VAEEncode -> KSampler`

同时，参考尺寸节点输出的图像接到 `Krea2洗图重平衡（肤质安全）`，用于帮助保构图。

修改本插件后需要重启 ComfyUI。旧浏览器页面如果仍显示旧参数名，可以按 `Ctrl+F5` 强制刷新。
